# python_sms

## Run Locally

Clone the project

```bash
  git clone https://github.com/ashutosh1808/python_sms.git
```

Go to the project directory

```bash
  cd python_sms
```

Install dependencies

```bash
  pip3 install matplotlib
```

```bash
  pip3 install requests
```

```bash
  pip3 install numpy
```
Start the server

```bash
  python sms.py
```
